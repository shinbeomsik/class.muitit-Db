[개념정리]
1. 정형데이터란?
저장할 데이터가 정해져 있는 데이터
항목이 정해진 데이터

2. DBMS란?
DataBase Management System
db와 관련된 일련의 작업을 가능하게 하는 프로그램(하드웨어를 포함하는 포괄적인 개념), 일반적으로 db 프로그램을 칭함
DB를 관리해주는 프로그램
.
3. 당신이 알고있는 DBMS의 종류 2가지 이상
오라클 mySQL

4. SQL은 무엇의 약자?
Structured Query Language 
체계적으로 정리된 명령문의 모음

5. SQL의 종류 3가지 분류
DDL DML  DCL

6. CRUD는 무엇의 약자?
Create Read Update Delete

7. 당신이 알고있는 DB클라이언트 프로그램은?
Dbeaver

8. DML중 데이터를 넣을 때 사용하는 구문은?
insert

9. DML중 데이터를 검색할 때 사용하는 구문은?
select
